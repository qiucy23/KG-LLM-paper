# RAG-Paper-Reading

A repository for knowledge graph related to RAG and LLM.

## 介绍 

这个仓库旨在记录和整理近年KG与LLM/RAG/Chain-of-Thought领域的前沿论文。

并非一个完全的论文收集仓库，而是一个论文学习仓库。因此会附上部分自己的理解。
